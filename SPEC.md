# SPEC.md - ローン繰上返済シミュレーター

## サイト概要

- **サイト名**: 繰上返済シミュレーター（仮）
- **目的**: 繰上返済による利息削減額・期間短縮を具体的な数字で見せる
- **収益モデル**: AdSense（金融系高単価） + 住宅ローン借り換え比較サイトへのアフィリエイト
- **技術構成**: React静的サイト、Cloudflare Pages
- **外部連携**: GitHub連携 → Cloudflare Pagesデプロイ、AdSense、Google Analytics、Search Console

## 計算ツール一覧

### 1. 繰上返済シミュレーター（メイン） `/kuriage`

**入力:**
- 借入額（万円）
- 年利（%）
- 返済期間（年）
- 返済方式：元利均等 / 元金均等
- 繰上返済額（万円）
- 繰上返済のタイミング（借入から何年後）
- 繰上返済の種類：期間短縮型 / 返済額軽減型

**出力:**
- 繰上返済なしの場合の総支払額・利息総額
- 繰上返済ありの場合の総支払額・利息総額
- 削減できる利息額
- 期間短縮型：短縮される期間
- 返済額軽減型：軽減後の月額
- **両タイプの比較表（並べて表示）** ← 差別化ポイント
- 月別の返済スケジュール表（折りたたみ）

### 2. 複数回繰上返済シミュレーション `/kuriage-multi`

- メインの拡張版。繰上返済を複数回設定できる
- 例：3年後に100万、7年後に200万
- 出力は同じだが、各回の効果の内訳も表示

### 3. 借り換えシミュレーター `/karikae`

**入力:**
- 現在のローン残高・残期間・金利
- 借り換え先の金利
- 借り換え諸費用（登記費用、事務手数料など。デフォルト値を用意）

**出力:**
- 借り換え後の月額・総支払額
- 諸費用込みでの損益分岐点（何年で元が取れるか）
- 「借り換えた方が得 / 損」の判定

### 4. ローン比較ツール `/hikaku`

- 2つのローン条件を並べて入力
- 月額・総支払額・利息総額を横並びで比較
- 固定金利 vs 変動金利の概算比較にも使える（変動は想定金利を入力）

## 計算ロジック

### 元利均等返済
```
月利 = 年利 / 12
月額 = 借入額 × 月利 × (1 + 月利)^返済月数 / ((1 + 月利)^返済月数 - 1)
```

### 元金均等返済
```
月の元金 = 借入額 / 返済月数
月の利息 = 残高 × 月利
月額 = 月の元金 + 月の利息（毎月変動）
```

### 繰上返済の再計算
- 指定タイミング時点の残高から繰上額を差し引く
- **期間短縮型**: 月額据え置きで期間を縮める
- **返済額軽減型**: 残期間据え置きで月額を再計算する

## 差別化ポイント

1. 期間短縮型と返済額軽減型を**常に並べて比較**できる
2. **複数回の繰上返済**に対応
3. 借り換えの**諸費用込みの損益分岐**を出す
4. 返済スケジュールの**月別表とグラフ**で実感しやすく
5. 住宅ローン以外（奨学金、車、カードローン）でも使えるよう**汎用的なラベル**

## 免責事項（必須）

- 概算であり、実際の返済額は金融機関の計算方法により異なる
- 税控除（住宅ローン控除）は考慮していない
- 変動金利の将来の金利変動は予測できない
- 金融商品の推奨ではない
